## Default Colorscheme

**OneDark** is set as the default colorscheme with a custom modification for visual selections.

**Configuration:** `nvim/lua/plugins/colorscheme.lua:9-22`

```lua
{
  "navarasu/onedark.nvim",
  opts = {
    highlights = {
      Visual = { bg = "#ffff00" },
    },
  },
},
{
  "LazyVim/LazyVim",
  opts = {
    colorscheme = "onedark",
  },
}
```

<Note>
The custom configuration sets visual selection highlight to bright yellow (`#ffff00`) for better visibility.
</Note>

## Switching Colorschemes

### Method 1: Permanently Change (Recommended)

Edit the colorscheme configuration file:

**File:** `nvim/lua/plugins/colorscheme.lua:18-22`

```lua
{
  "LazyVim/LazyVim",
  opts = {
    colorscheme = "kanagawa", -- Change this to your preferred theme
  },
}
```

Available options: `onedark`, `kanagawa`, `gruvbox`, `rose-pine`, `tokyodark`, `everforest`, `PaperColor`

<Steps>
  <Step title="Edit the config file">
    Open `~/.config/nvim/lua/plugins/colorscheme.lua`
  </Step>
  
  <Step title="Change colorscheme value">
    Update the `colorscheme` field to your desired theme name
  </Step>
  
  <Step title="Restart Neovim">
    Close and reopen Neovim or run `:source %` if editing the config file
  </Step>
  
  <Step title="Verify the change">
    The new colorscheme should be applied automatically
  </Step>
</Steps>

### Method 2: Try Temporarily

Test colorschemes without editing files:

```vim
:colorscheme kanagawa
:colorscheme gruvbox
:colorscheme rose-pine
:colorscheme tokyodark
:colorscheme everforest
:colorscheme PaperColor
:colorscheme onedark
```

<Note>
This change is temporary and will reset when you restart Neovim.
</Note>

### Method 3: Interactive Selection

Use Telescope to preview and select colorschemes:

```vim
:Telescope colorscheme
```

This opens a live preview where you can:
- Navigate through colorschemes with `<Up>` and `<Down>`
- See real-time preview as you move
- Press `<Enter>` to apply (temporary until you update config)

## Customizing Colorschemes

### OneDark Customization

The OneDark theme includes custom highlight modifications. You can add more:

```lua
{
  "navarasu/onedark.nvim",
  opts = {
    style = "darker", -- Options: dark, darker, cool, deep, warm, warmer
    highlights = {
      Visual = { bg = "#ffff00" },
      -- Add more custom highlights
      ["@comment"] = { fg = "#7a8a99", italic = true },
      LineNr = { fg = "#4b5263" },
    },
    colors = {
      -- Override specific colors
      bright_orange = "#ff8800",
    },
  },
}
```

### Adding Transparent Background

For any colorscheme, you can enable transparency:

```lua
{
  "navarasu/onedark.nvim",
  opts = {
    transparent = true,
    highlights = {
      Visual = { bg = "#ffff00" },
    },
  },
}
```

<Warning>
Transparent backgrounds require terminal support and may not work in all terminal emulators.
</Warning>

## Adding New Colorschemes

To add additional colorschemes:

<Steps>
  <Step title="Add plugin to config">
    Edit `nvim/lua/plugins/colorscheme.lua` and add the plugin:
    
    ```lua
    return {
      -- Existing colorschemes...
      
      -- Add new colorscheme
      { "catppuccin/nvim", name = "catppuccin" },
    }
    ```
  </Step>
  
  <Step title="Restart Neovim">
    Lazy.nvim will automatically install the new plugin
  </Step>
  
  <Step title="Test the colorscheme">
    Try it with `:colorscheme catppuccin`
  </Step>
  
  <Step title="Make it default (optional)">
    Update the LazyVim opts to use the new colorscheme
  </Step>
</Steps>

## Tips for Choosing a Colorscheme

<AccordionGroup>
  <Accordion title="Consider Your Environment">
    - **Bright rooms**: Light themes like PaperColor or Rose Pine Dawn
    - **Dark rooms**: Dark themes like Tokyo Dark or OneDark
    - **Mixed**: Choose themes with light/dark variants like Everforest or Gruvbox
  </Accordion>
  
  <Accordion title="Eye Strain Considerations">
    - Lower contrast themes: Kanagawa, Everforest
    - Higher contrast themes: OneDark, Gruvbox
    - Warm tones: Gruvbox, Everforest
    - Cool tones: OneDark, Tokyo Dark
  </Accordion>
  
  <Accordion title="Programming Language">
    Some themes have better support for specific languages:
    - **JavaScript/TypeScript**: OneDark, Tokyo Dark
    - **Python**: Gruvbox, Everforest
    - **All languages**: LazyVim themes are well-tested across languages
  </Accordion>
  
  <Accordion title="Personal Preference">
    Try each theme for a day to see which one feels most comfortable. Use `:Telescope colorscheme` to quickly test them all.
  </Accordion>
</AccordionGroup>

<Tip>
The colorscheme you choose can significantly impact your productivity and comfort. Don't hesitate to switch themes seasonally or based on time of day.
</Tip>
