## Prefix Key

<Note>
  This configuration uses `Ctrl+Space` as the prefix key instead of the default `Ctrl+b`.
</Note>

```bash
# Custom prefix configuration
unbind C-b
set -g prefix C-Space
bind C-Space send-prefix
```

All keybindings below require pressing the **Prefix** key first, unless otherwise noted.

<Tabs>
  <Tab title="Prefix: Ctrl+Space">
    Press `Ctrl+Space`, release, then press the command key.
    
    Example: To create a new window, press `Ctrl+Space` then `c`.
  </Tab>
  
  <Tab title="Sending Prefix to Applications">
    To send the actual `Ctrl+Space` keystroke to a program running in tmux (like Vim), press `Ctrl+Space` twice.
  </Tab>
</Tabs>

## Custom Keybindings

These are keybindings explicitly configured in `tmux.conf`:

### Copy Mode Entry

| Keybinding | Action | Configuration |
|------------|--------|---------------|
| `Prefix + [` | Enter copy mode | `bind-key [ copy-mode` |

```bash
# Enter copy mode with Prefix+[
bind-key [ copy-mode
```

### Copy Mode Keybindings (Vim-style)

These keybindings work when in copy mode:

| Keybinding | Action | Configuration |
|------------|--------|---------------|
| `v` | Begin selection (visual mode) | `bind-key -T copy-mode-vi 'v' send -X begin-selection` |
| `y` | Copy selection (yank) | `bind-key -T copy-mode-vi 'y' send -X copy-selection` |
| `V` | Toggle rectangle selection | `bind-key -T copy-mode-vi 'V' send -X rectangle-toggle` |

```bash
# Vim-like navigation in copy mode
bind-key -T copy-mode-vi 'v' send -X begin-selection
bind-key -T copy-mode-vi 'y' send -X copy-selection
bind-key -T copy-mode-vi 'V' send -X rectangle-toggle
```

<Accordion title="Copy Mode Workflow Example">
  1. Press `Ctrl+Space` then `[` to enter copy mode
  2. Navigate using Vim keys (`h`, `j`, `k`, `l`, `w`, `b`, etc.)
  3. Press `v` to start visual selection
  4. Navigate to extend selection
  5. Press `y` to copy (yank) the selected text
  6. Press `q` or `Esc` to exit copy mode
  7. Paste with `Prefix + ]`
</Accordion>

## Default Tmux Keybindings

These are standard tmux keybindings that work with the custom `Ctrl+Space` prefix:

### Session Management

| Keybinding | Action | Description |
|------------|--------|-------------|
| `Prefix + d` | Detach from session | Detach from current session (session continues running) |
| `Prefix + $` | Rename session | Rename the current session |
| `Prefix + s` | List sessions | Show all sessions and switch between them |
| `Prefix + (` | Previous session | Switch to previous session |
| `Prefix + )` | Next session | Switch to next session |

### Window Management

| Keybinding | Action | Description |
|------------|--------|-------------|
| `Prefix + c` | Create window | Create a new window |
| `Prefix + ,` | Rename window | Rename the current window |
| `Prefix + &` | Kill window | Close current window (with confirmation) |
| `Prefix + n` | Next window | Switch to next window |
| `Prefix + p` | Previous window | Switch to previous window |
| `Prefix + 0-9` | Switch to window | Switch to window by number (1-9, 0) |
| `Prefix + l` | Last window | Switch to last active window |
| `Prefix + w` | List windows | Show window list and switch interactively |
| `Prefix + f` | Find window | Search for window by name |

<Note>
  Windows start at index 1 (not 0) due to `set -g base-index 1` in the configuration.
</Note>

### Pane Management

| Keybinding | Action | Description |
|------------|--------|-------------|
| `Prefix + %` | Split horizontally | Split current pane vertically (side by side) |
| `Prefix + "` | Split vertically | Split current pane horizontally (top and bottom) |
| `Prefix + x` | Kill pane | Close current pane (with confirmation) |
| `Prefix + z` | Toggle pane zoom | Maximize/restore current pane |
| `Prefix + !` | Break pane | Convert pane to new window |
| `Prefix + q` | Show pane numbers | Display pane numbers (press number to switch) |
| `Prefix + {` | Move pane left | Swap current pane with previous |
| `Prefix + }` | Move pane right | Swap current pane with next |
| `Prefix + Space` | Cycle layouts | Cycle through pane layouts |
| `Prefix + o` | Next pane | Switch to next pane |
| `Prefix + ;` | Last pane | Switch to last active pane |

### Pane Navigation

| Keybinding | Action | Description |
|------------|--------|-------------|
| `Prefix + ↑` | Move up | Switch to pane above |
| `Prefix + ↓` | Move down | Switch to pane below |
| `Prefix + ←` | Move left | Switch to pane on the left |
| `Prefix + →` | Move right | Switch to pane on the right |

<Warning>
  The arrow key navigation uses the custom `Ctrl+Space` prefix, not the default `Ctrl+b`.
</Warning>

### Pane Resizing

| Keybinding | Action | Description |
|------------|--------|-------------|
| `Prefix + Ctrl + ↑` | Resize up | Increase pane height upward |
| `Prefix + Ctrl + ↓` | Resize down | Increase pane height downward |
| `Prefix + Ctrl + ←` | Resize left | Increase pane width leftward |
| `Prefix + Ctrl + →` | Resize right | Increase pane width rightward |
| `Prefix + Alt + ↑` | Resize up (large) | Increase pane height upward by 5 |
| `Prefix + Alt + ↓` | Resize down (large) | Increase pane height downward by 5 |
| `Prefix + Alt + ←` | Resize left (large) | Increase pane width leftward by 5 |
| `Prefix + Alt + →` | Resize right (large) | Increase pane width rightward by 5 |

## Plugin Keybindings

TPM (Tmux Plugin Manager) adds these keybindings:

| Keybinding | Action | Description |
|------------|--------|-------------|
| `Prefix + I` | Install plugins | Install plugins listed in config (capital I) |
| `Prefix + U` | Update plugins | Update all installed plugins |
| `Prefix + Alt + u` | Uninstall plugins | Remove plugins not in config |

```bash
# Plugin management
set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'dreamsofcode-io/catppuccin-tmux'
```

## Adding Custom Keybindings

You can add your own keybindings to `tmux.conf`:

### Example: Reload Configuration

```bash
# Reload config with Prefix + r
bind r source-file ~/.config/tmux/tmux.conf \; display "Config reloaded!"
```

### Example: Split Panes with Intuitive Keys

```bash
# Split with | and -
bind | split-window -h
bind - split-window -v
unbind '"'
unbind %
```

### Example: Vim-like Pane Navigation

```bash
# Use h, j, k, l for pane navigation
bind h select-pane -L
bind j select-pane -D
bind k select-pane -U
bind l select-pane -R
```

### Example: Vim-like Pane Resizing

```bash
# Resize panes with H, J, K, L
bind -r H resize-pane -L 5
bind -r J resize-pane -D 5
bind -r K resize-pane -U 5
bind -r L resize-pane -R 5
```

<Note>
  The `-r` flag makes the keybinding repeatable, so you can press the prefix once and then press the key multiple times.
</Note>

```bash tmux.conf (keybindings only)
# Prefix key: Ctrl+Space
unbind C-b

```

## Keybinding Conflicts

<Warning>
  Be aware of potential keybinding conflicts with other applications:
</Warning>

<Tabs>
  <Tab title="Vim/Neovim">
    If you use Vim inside tmux, `Ctrl+Space` won't conflict as it's not a common Vim keybinding. However, ensure you can send `Ctrl+Space` twice if needed.
  </Tab>
  
  <Tab title="Terminal Emulator">
    Some terminal emulators may intercept `Ctrl+Space`. Test that tmux receives the keybinding correctly.
  </Tab>
  
  <Tab title="Shell">
    `Ctrl+Space` is typically not used by shells (bash, zsh, fish), so conflicts are rare.
  </Tab>
</Tabs>
