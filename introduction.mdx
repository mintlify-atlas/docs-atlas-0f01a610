## Welcome to Your Ultimate Development Setup

This is a carefully crafted collection of configuration files designed to create a seamless, efficient, and aesthetically pleasing development environment on macOS. Built around powerful tools like Neovim with LazyVim, Tmux, Alacritty, and AeroSpace, this setup emphasizes keyboard-driven workflows and minimal distractions.

## Key Features

### Neovim Configuration

Built on top of the LazyVim distribution, this Neovim setup includes:

- **Rich Color Scheme Collection**: Multiple themes including OneDark (default), Kanagawa, Gruvbox, Rose Pine, TokyoDark, Everforest, and PaperColor
- **Obsidian Integration**: Native markdown editing with vault synchronization for note-taking workflows
- **LeetCode Plugin**: Practice coding problems directly within Neovim with Python3 support
- **Custom Tab Completion**: Enhanced autocomplete experience tailored to your workflow
- **Snacks Configuration**: Additional UI enhancements and utilities
- **Vim Be Good**: Practice and improve your Vim motions

### Tmux Setup

A productivity-focused terminal multiplexer configuration featuring:

- **Catppuccin Theme**: Beautiful, eye-friendly color scheme from Dreams of Code
- **Space Prefix**: Remapped from `Ctrl+b` to `Ctrl+Space` for easier access
- **Vim Keybindings**: Full Vi-mode support in copy mode with familiar `v`, `y`, `V` selections
- **True Color Support**: 256-color and RGB color support for modern terminals
- **Smart Window Management**: Auto-renumbering windows, 1-based indexing, mouse support
- **TPM Plugin Manager**: Easy plugin installation and updates

### Alacritty Terminal

A blazingly fast GPU-accelerated terminal with:

- **0xProto Nerd Font**: Monospace font with programming ligatures and icon support at 16pt
- **Cool Night Theme**: Custom color scheme for comfortable long coding sessions
- **Visual Effects**: 80% opacity with blur effects for a modern aesthetic
- **Generous Padding**: 10px horizontal and vertical padding for better readability
- **Buttonless Decorations**: Clean, distraction-free window chrome on macOS
- **Maximized Startup**: Opens full-screen ready for immediate productivity
- **Extensive Theme Collection**: Includes 150+ pre-configured themes (Tokyo Night, Nord, Solarized, Rose Pine, and more)

### AeroSpace Window Manager

A powerful i3-like tiling window manager designed specifically for macOS:

- **Vim-Style Navigation**: Use `Alt+h/j/k/l` to move focus between windows
- **Extensive Workspace Support**: 9 numbered workspaces plus 26 letter-named workspaces (A-Z)
- **Multiple Layout Modes**: Tiles (horizontal/vertical) and accordion layouts
- **Mouse Follows Focus**: Automatic cursor positioning for seamless navigation
- **Smart Window Management**: Automatic container normalization and orientation
- **Service Mode**: Advanced operations like config reload, layout reset, floating toggle
- **Zero Gaps**: Clean edge-to-edge window arrangements
- **Auto-start on Login**: Launches automatically with your macOS session

## Philosophy

This configuration emphasizes:

- **Keyboard-First Workflow**: Minimize mouse usage with comprehensive keybindings
- **Consistent Vim Keybindings**: Vim motions across all tools for muscle memory
- **Visual Harmony**: Coordinated color schemes and aesthetics across all applications
- **Performance**: Fast, efficient tools that don't slow you down
- **Extensibility**: Well-organized configuration files that are easy to customize
- **Productivity**: Thoughtful defaults that reduce friction in daily development tasks
