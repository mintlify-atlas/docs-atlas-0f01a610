This configuration uses LazyVim's default keymaps with minimal custom overrides. LazyVim provides a comprehensive set of keybindings designed for efficient text editing and navigation.

## Custom Keymaps

**File:** `nvim/lua/config/keymaps.lua`

This configuration intentionally keeps custom keymaps minimal, relying on LazyVim's well-designed defaults:

```lua
-- Keymaps are automatically loaded on the VeryLazy event
-- Default keymaps that are always set: https://github.com/LazyVim/LazyVim/blob/main/lua/lazyvim/config/keymaps.lua
-- Add any additional keymaps here
```

<Note>
No custom keymaps are defined. This configuration uses 100% of LazyVim's default keybindings.
</Note>

## Tmux Integration

If you're using Tmux alongside Neovim, window navigation keymaps (`<C-h/j/k/l>`) work seamlessly across Tmux panes and Neovim windows when properly configured.

## Learning More

To discover all available keymaps in real-time:

```vim
:Telescope keymaps
```

Or use Which-key (included in LazyVim) - just press `<Space>` and wait a moment to see available keybindings.

## Adding Custom Keymaps

To add your own keymaps, edit `nvim/lua/config/keymaps.lua`:

```lua
-- Example custom keymaps
local map = vim.keymap.set

-- Map jj to exit insert mode
map("i", "jj", "<Esc>", { desc = "Exit insert mode" })

-- Quick save with Ctrl+S
map({ "n", "i", "v" }, "<C-s>", "<cmd>w<cr><esc>", { desc = "Save file" })

-- Better navigation in command mode
map("c", "<C-h>", "<Left>", { desc = "Move left" })
map("c", "<C-l>", "<Right>", { desc = "Move right" })
```
