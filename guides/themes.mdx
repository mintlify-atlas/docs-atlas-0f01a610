## Alacritty Themes

Alacritty comes with an extensive collection of pre-configured themes located in `alacritty/themes/themes/`.

### Current Theme

The configuration currently uses the **Coolnight** theme:

```toml
general.import = [
    "~/.config/alacritty/themes/themes/coolnight.toml"
]
```

### Switching Alacritty Themes

<Steps>

<Step title="Browse Available Themes">

List all available themes:

```bash
ls alacritty/themes/themes/
```

Or view them in your file explorer.

</Step>

<Step title="Choose Your Theme">

Pick a theme file from the list, for example:
- `tokyo_night.toml`
- `catppuccin_mocha.toml`
- `gruvbox_dark.toml`

</Step>

<Step title="Update Configuration">

Edit `alacritty/alacritty.toml` and change the import line:

```toml
general.import = [
    "~/.config/alacritty/themes/themes/tokyo_night.toml"
]
```

Replace `tokyo_night.toml` with your chosen theme.

</Step>

<Step title="Reload Alacritty">

Alacritty automatically reloads the configuration. If it doesn't, close and reopen Alacritty, or:

- macOS: `Cmd + ,` to reload config
- Linux: Configuration reloads automatically

</Step>

</Steps>

<Note>
You can preview themes by temporarily changing the import line and immediately seeing the result.
</Note>

## Neovim Colorschemes

The Neovim configuration includes multiple colorschemes with OneDark set as default.

### Installed Colorschemes

**File:** `nvim/lua/plugins/colorscheme.lua`

```lua
return {
  { "rebelot/kanagawa.nvim" },
  { "NLKNguyen/papercolor-theme" },
  { "ellisonleao/gruvbox.nvim" },
  { "rose-pine/neovim" },
  { "tiagovla/tokyodark.nvim" },
  { "sainnhe/everforest" },
  {
    "navarasu/onedark.nvim",
    opts = {
      highlights = {
        Visual = { bg = "#ffff00" },
      },
    },
  },
  {
    "LazyVim/LazyVim",
    opts = {
      colorscheme = "onedark",
    },
  },
}
```

### Available Neovim Colorschemes

| Colorscheme | Command | Description |
|-------------|---------|-------------|
| **OneDark** (default) | `:colorscheme onedark` | Atom's dark theme with yellow visual selection |
| **Kanagawa** | `:colorscheme kanagawa` | Inspired by the famous painting |
| **PaperColor** | `:colorscheme papercolor` | Clean light/dark theme |
| **Gruvbox** | `:colorscheme gruvbox` | Retro groove color scheme |
| **Rose Pine** | `:colorscheme rose-pine` | Natural pine with soft colors |
| **TokyoDark** | `:colorscheme tokyodark` | Clean dark theme |
| **Everforest** | `:colorscheme everforest` | Green forest theme |

### Switching Neovim Colorschemes

#### Temporary Change (Current Session)

Inside Neovim, use the colorscheme command:

```vim
:colorscheme kanagawa
```

#### Permanent Change

Edit `nvim/lua/plugins/colorscheme.lua` and change the default:

```lua
{
  "LazyVim/LazyVim",
  opts = {
    colorscheme = "kanagawa",  -- Change this
  },
}
```

Restart Neovim or run:

```vim
:Lazy reload colorscheme
```

### Colorscheme Customization

The OneDark colorscheme has a custom visual selection highlight:

```lua
{
  "navarasu/onedark.nvim",
  opts = {
    highlights = {
      Visual = { bg = "#ffff00" },  -- Bright yellow background
    },
  },
}
```

You can add similar customizations to other colorschemes:

```lua
{
  "rebelot/kanagawa.nvim",
  opts = {
    overrides = function(colors)
      return {
        -- Custom overrides here
        Visual = { bg = colors.palette.waveBlue2 },
      }
    end,
  },
}
```

### Adding New Neovim Colorschemes

To add a new colorscheme:

1. Add the plugin to `colorscheme.lua`:

```lua
return {
  -- Existing colorschemes...
  { "folke/tokyonight.nvim" },  -- New colorscheme
}
```

2. Restart Neovim or run `:Lazy sync`

3. Switch to it:

```vim
:colorscheme tokyonight
```

## Tmux Theme

Tmux uses the Catppuccin theme for a cohesive experience.

### Current Configuration

**File:** `tmux/tmux.conf`

```bash
set -g @plugin 'dreamsofcode-io/catppuccin-tmux'
```

### Installing the Theme

The Catppuccin theme is installed via TPM (Tmux Plugin Manager):

```
Ctrl+Space + I
```

### Switching Tmux Themes

To use a different Tmux theme:

<Steps>

<Step title="Find a Theme">

Browse Tmux themes:
- [Tmux Plugins](https://github.com/tmux-plugins)
- [Awesome Tmux](https://github.com/rothgar/awesome-tmux)

Popular themes:
- `catppuccin/tmux` - Official Catppuccin
- `dracula/tmux` - Dracula theme
- `egel/tmux-gruvbox` - Gruvbox theme
- `jimeh/tmux-themepack` - Collection of themes
- `rose-pine/tmux` - Rose Pine theme

</Step>

<Step title="Update Configuration">

Edit `~/.tmux.conf`:

```bash
# Remove or comment out current theme
# set -g @plugin 'dreamsofcode-io/catppuccin-tmux'

# Add new theme
set -g @plugin 'dracula/tmux'
```

</Step>

<Step title="Install Theme">

Inside Tmux, install the new theme:

```
Ctrl+Space + I
```

</Step>

<Step title="Reload Configuration">

Reload Tmux configuration:

```bash
tmux source ~/.tmux.conf
```

Or use the keybinding:

```
Ctrl+Space + r
```

</Step>

</Steps>

### Catppuccin Variants

Catppuccin has four variants you can configure:

```bash
set -g @catppuccin_flavour 'mocha'     # Options: latte, frappe, macchiato, mocha
```

Add this line before the TPM initialization in your `~/.tmux.conf`.

## Troubleshooting

### Colors Look Wrong

**Issue:** Colors don't match the theme preview

**Solution:**
1. Ensure your terminal supports true color (24-bit):
   ```bash
   echo $TERM
   ```
   Should show `xterm-256color` or similar

2. Check Alacritty configuration has:
   ```toml
   [env]
   TERM = "xterm-256color"
   ```

### Neovim Theme Not Loading

**Issue:** Colorscheme command fails

**Solution:**
1. Run `:Lazy sync` to ensure plugins are installed
2. Check for errors: `:Lazy log`
3. Verify plugin is loaded: `:Lazy`

### Tmux Theme Not Appearing

**Issue:** Theme doesn't change after installation

**Solution:**
1. Kill all Tmux sessions: `tmux kill-server`
2. Start new session: `tmux`
3. Verify plugin is installed: `ls ~/.tmux/plugins/`
